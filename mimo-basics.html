Transmit Diversity vs. Receive Diversity

Receive diversity is achieved by configuring multiple antennas in receiver side. Each receiver antenna receives an independently faded copy of the transmitted symbol.

Then combining technique is employed to get an estimated version of the original symbol. There are several kinds of combining method, like selective combining (SC), equal gain combining (EGC), maximum ratio combining (MRC).

Receive diversity can be easily implemented and optimal performance will be obtained with maximum ratio combining. It is suited for the uplink scenarios, i. e. equip the base station tower with multiple antennas. Obviously, multiple antennas need much more space to configure and require more powerful computing unit to do the complicated receive algorithms which means more cost.

So what about downlink scenario? Mobile terminals is always small and cheap, and it’s not practical to equip mobile terminals (like smart phones) with multiple antennas. To cope with this, maybe we can configure multiple antennas in the transmitter side instead, and transmit the same symbol from these antennas.

Someone may immediately come up with an easy idea that utilises $N_T$ transmit antennas transmit the same symbols simultaneously, but simple mathematics will show that there’s no diversity here. This is the motivation of Space Time Coding (STC).

Maybe there is someone still confused by the difference between multiple transmit antennas and multiple receive antennas to achieve diversity. What causes this is that channel state information (CSI) can be estimated at the receiver easily by kinds of channel estimation algorithms while CSI is not available at transmitter side in most cases.

It is easy to see that if we know CSI at transmitter side, let’s say $h_i=\alpha_i\cdot e^{j\theta}$, We can multiply the symbol from i-th antenna with $e^{-j\theta}$ and a normalisation factor which matters little. After propagating across the multiple paths, different version of transmitted symbol are co-phased. By elaborately choosing the normalised factor, the same performance of receive diversity with MRC can be achieved.
